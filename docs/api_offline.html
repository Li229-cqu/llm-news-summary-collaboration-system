后端 API 接口总览

统一整理本项目后端接口，按模块分组展示。

说明：当前项目接口统一返回 code / message / data 结构。
AUTH认证模块 登录、退出、权限校验

认证模块采用 mock token 方案。

方法路径登录说明返回要点
POST/api/auth/login否账号密码登录token、user
POST/api/auth/logout是退出登录清空登录状态
GET/api/auth/me是获取当前用户信息user、role、avatar
GET/api/auth/check-login是检查是否登录登录态校验
GET/api/auth/check-editor是检查 editor 权限401 / 403 / success
GET/api/auth/check-admin是检查 admin 权限401 / 403 / success
NEWS新闻模块 列表、详情、热榜、搜索、浏览记录

新闻模块当前数据库优先，数据库异常时回退 mock。新闻详情与列表允许未登录访问。

方法路径登录说明返回要点
GET/api/news/categories否获取新闻分类分类列表
GET/api/news否获取新闻列表list、total、page
GET/api/news/hot否获取新闻热榜 Top10rank、view_count
GET/api/news/search否搜索新闻分页结果
GET/api/news/{news_id}否获取新闻详情正文、相关文章、推荐阅读
POST/api/news/{news_id}/browse否记录浏览行为recorded=true
INTERACTION新闻互动模块 点赞、收藏、评论、回复、评论点赞

互动写操作需要登录；未登录返回 401。评论列表允许未登录查看。

方法路径登录说明返回要点
POST/api/news/{news_id}/like是点赞新闻liked、like_count
DELETE/api/news/{news_id}/like是取消点赞新闻liked、like_count
POST/api/news/{news_id}/favorite是收藏新闻favorited、favorite_count
DELETE/api/news/{news_id}/favorite是取消收藏新闻favorited、favorite_count
GET/api/news/{news_id}/comments否获取评论列表list、total、replies
POST/api/news/{news_id}/comments是发布一级评论comment_count 增加
POST/api/comments/{comment_id}/reply是回复评论parent_id、news_id
POST/api/comments/{comment_id}/like是点赞评论liked、like_count
PROFILE个人中心模块 浏览、收藏、评论、AI 记录、订阅、阅读脉络

个人中心数据优先从数据库读取，数据库异常时保留 mock fallback。

方法路径登录说明返回要点
GET/api/profile/overview是个人中心概览browse_count、favorite_count 等
GET/api/profile/browse-history是浏览历史title、category_name、browse_time
GET/api/profile/favorites是收藏记录title、summary、source
GET/api/profile/comments是评论记录news_title、content、create_time
GET/api/profile/ai-records是AI 生成记录source、summary_short、risk_level
GET/api/profile/subscriptions是分类订阅状态categories、subscribed_category_ids
POST/api/profile/subscriptions是更新分类订阅更新结果
GET/api/profile/reading-trajectory是阅读脉络图summary、nodes、edges
GET/api/profile/reading-timeline是阅读时间线summary、items
GET/api/profile/reading-heatmap是阅读热力图x_axis、y_axis、cells、summary
TIMELINETimeline 模块 多源事件脉络

Timeline 当前数据库优先，ai-service 兜底；前端以抽屉方式嵌入首页、社区和新闻详情页展示。

方法路径登录说明返回要点
GET/api/timeline/topics否话题列表topic_id、topic_name、heat_score
GET/api/timeline/topics/{topic_id}/news否话题相关新闻按发布时间升序
GET/api/timeline/topics/{topic_id}否获取时间线timeline、source、generate_status
POST/api/timeline/topics/{topic_id}/generate是生成时间线timeline、generated_at
COMMUNITY社区模块 帖子、评论、点赞、收藏、热搜

社区模块当前数据库优先，数据库不可用时回退 mock。

方法路径登录说明返回要点
GET/api/community/posts否帖子列表page、page_size、keyword
GET/api/community/posts/{post_id}否帖子详情post、comments
POST/api/community/posts是发布帖子id、title、content
POST/api/community/posts/{post_id}/like是点赞帖子liked、like_count
DELETE/api/community/posts/{post_id}/like是取消点赞帖子liked、like_count
POST/api/community/posts/{post_id}/favorite是收藏帖子favorited、favorite_count
DELETE/api/community/posts/{post_id}/favorite是取消收藏帖子favorited、favorite_count
ADMIN管理后台 概览、用户、待审核内容

管理后台接口数据库优先，数据库异常时可回退 mock。

方法路径登录说明返回要点
GET/api/admin/ping否健康检查ok
GET/api/admin/dashboard是后台概览用户数、新闻数、待审数
GET/api/admin/users是用户列表不返回 password
GET/api/admin/pending-posts是待审核帖子分页列表
GET/api/admin/system-config是系统配置配置项
AIAI 服务模块 标题摘要、记录、健康检查

AI 服务当前以 mock 结果为主，后端优先调用 ai-service，失败时回退本地规则。

方法路径登录说明返回要点
GET/api/ai/health否AI 服务健康检查status、service
POST/api/ai/generate是生成标题与摘要candidate_titles、summary_short、summary_long
GET/api/ai/records是AI 生成记录分页记录
POST/api/ai/generate-timeline否Timeline 生成timeline、topic_name
