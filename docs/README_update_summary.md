#### 8.1 .env 文件处理

```
【重要】：ai-service/.env 包含敏感信息（API Key），不会被提交到 Git。

使用流程：
1. 自动生成的 .env.example 包含配置模板
2. 克隆项目后，复制 .env.example 为 .env
3. 编辑 .env，填入你的 API Key（可选，默认使用 Mock）
4. .env 已在 .gitignore 中，不会被提交
```
