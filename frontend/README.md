# 基于大语言模型的智能新闻摘要与协同互动系统：前端工程

## 一、前端项目说明

`frontend` 是基于大语言模型的智能新闻摘要与协同互动系统的前端工程。

当前处于第 1 阶段：前端基础框架搭建。本阶段已完成工程初始化、路由、状态管理、统一布局、主题切换和基础页面占位，为后续模块开发提供统一的前端基础。

## 二、技术栈

- Vue 3
- Vite
- TypeScript
- Vue Router
- Pinia
- Element Plus
- Axios

## 三、目录结构说明

```text
src/
├── api/                    # Axios 请求实例与后续接口封装
├── assets/                 # 静态资源
├── components/             # 公共组件
│   ├── layout/             # 布局组件，如顶部栏与左侧栏
│   └── common/             # 通用组件
├── layouts/                # 页面整体布局
├── router/                 # 路由配置
├── stores/                 # Pinia 状态管理
├── styles/                 # CSS 变量与全局样式
├── utils/                  # 工具函数
└── views/                  # 页面组件
```

## 四、当前已完成内容

1. Vue 3 项目初始化。
2. 路由配置。
3. Pinia 状态管理。

## 五、运行方式

在 `frontend` 目录执行：

```bash
npm install
npm run dev
```

启动成功后，根据终端输出的本地地址访问项目，通常为 `http://localhost:5173`。

## 六、当前注意事项

1. 当前不连接后端。
2. 当前不连接数据库。
3. 当前登录基于后端 Mock 用户，不连接数据库。
4. 当前不实现真实新闻、AI、社区业务。
5. 所有页面仍以基础框架和占位内容为主。

## 七、用户与权限 Mock

当前已支持 Mock 登录、用户状态管理、路由守卫、角色权限控制，以及顶部导航栏的登录状态展示。

测试账号：

| 角色 | 用户名 | 密码 |
| --- | --- | --- |
| 普通用户 | `user` | `123456` |
| 审核编辑 | `editor` | `123456` |
| 管理员 | `admin` | `123456` |

普通用户登录后可访问需要登录的页面；审核编辑和管理员可访问管理后台，管理员可查看管理员专属入口。
