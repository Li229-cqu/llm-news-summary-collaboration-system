# 《基于大语言模型的智能新闻摘要与协同互动系统》前端工程

## 一、前端项目说明

`frontend` 是本系统的前端工程，当前处于新闻模块持续完善阶段。页面整体采用“顶部导航栏 + 左侧分类栏 + 主内容区”的布局方式，新闻首页、新闻详情页与互动功能已经接入 mock 后端接口。

## 二、技术栈

- Vue 3
- Vite
- TypeScript
- Vue Router
- Pinia
- Element Plus
- Axios

## 三、目录结构说明

```text
src/
├─ api/                # 接口封装
├─ assets/             # 静态资源
├─ components/         # 公共组件
│  ├─ layout/          # 顶部导航栏、侧边栏等布局组件
│  ├─ news/            # 新闻展示相关组件
│  └─ interaction/     # 点赞、收藏、评论等互动组件
├─ layouts/            # 页面整体布局
├─ router/             # 路由配置
├─ stores/             # Pinia 状态管理
├─ styles/             # CSS 变量与全局样式
├─ utils/              # 工具函数
└─ views/              # 页面视图
```

## 五、运行方式

```bash
npm install
npm run dev
```

启动成功后，通常访问：

```text
http://localhost:5173
```

## 六、当前注意事项

1. 当前不连接数据库。
2. 当前不直接调用 `ai-service`。
3. 当前不实现真实登录和真实密码加密。
4. 当前新闻、详情、互动数据来自 backend mock 接口。
5. 当前互动状态和评论数据为 mock 内存更新，服务重启后不会持久化。
6. 当前所有页面仍以基础框架和占位内容为主，真实业务会在后续阶段继续完善。

## 八、首页模块说明

首页当前已接入：

- 新闻分类
- 新闻列表
- 搜索
- 热榜

首页数据来自 backend mock 接口，不连接数据库，也不直接调用 `ai-service`。

首页交互说明：

- 点击新闻卡片跳转 `/news/:id`
- 点击 AI 工具入口跳转 `/ai/title-summary`

## 九、新闻详情页说明

新闻详情页当前已支持：

- 新闻详情展示
- 新闻浏览记录
- 相关文章与推荐阅读
- 跳转 AI 标题摘要生成页
- 点赞
- 取消点赞
- 收藏
- 取消收藏
- 评论列表展示
- 发布评论
- 回复评论
- 评论点赞

未登录用户进行互动时会跳转登录页。

当前互动数据来自 backend mock 接口，不连接数据库，互动状态重启后不会持久化。
